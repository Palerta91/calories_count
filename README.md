# Проект: предсказание калорийности блюд

Задача проекта — оценить калорийность блюда по трем источникам информации:

- изображению блюда;
- списку ингредиентов;
- массе порции.

Основная логика проекта собрана в ноутбуке `notebook_V1.3.ipynb`. В нем последовательно выполнены:

- исследовательский анализ данных;
- подготовка обучающего пайплайна;
- обучение модели;
- проверка качества на тестовой выборке;
- разбор самых сложных примеров.

Структура проекта:

- `notebook_V1.3.ipynb` — основной ноутбук проекта;
- `scripts/config.py` — параметры запуска и обучения;
- `scripts/dataset.py` — подготовка данных, датасет и преобразования;
- `scripts/utils.py` — модель, обучение, инференс и визуализация ошибок;
- `requirements.txt` — зависимости проекта.

Данные ожидаются в папке `data/`:

```text
data/
├── dish.csv
├── ingredients.csv
└── images/
    └── dish_.../
        └── rgb.png
```

Артефакт лучшей модели сохраняется в `artifacts/best_model.pt`.
